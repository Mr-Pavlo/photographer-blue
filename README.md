# Photographer Landing Page

Responsive landing page for a photographer's portfolio, created as one of my first frontend practice projects.

## Live Demo

[View the live website](https://mr-pavlo.github.io/photographer-blue/)

## Technologies

- HTML5
- SCSS
- CSS
- Flexbox
- Responsive Web Design
- GitHub Pages

## Features

- Semantic HTML structure
- Responsive layout
- Sections: Home, About, Portfolio, Pricing, Contacts
- Reusable SCSS variables and mixins
- Image collage and portfolio gallery
- Client reviews section
- Hover effects and transitions

## About the Project

This was one of my first landing pages, built to practice HTML5 and SCSS.
The code isn't fully clean, but the page is fully functional and responsive.
